🚀 海外达人营销自动化管理系统 (Influencer Tracker)
这是一个专为海外出海团队打造的极简、安全、端到端达人（Influencer）建联与跟进管理系统。

🌟 项目核心价值 (The "Why")
在海外红人营销过程中，团队常常面临数据碎片化、Excel 录入低效、跨团队协作困难以及跟进状态断层等痛点。该系统旨在通过 AI 辅助的自动化手段，实现：

10倍速录入：智能提取碎片化文本中的邮箱，彻底告别手动复制粘贴。

云端实时协作：数据统一下云端，解决多人操作 Excel 导致的版本冲突。

全链路跟进可视化：自动生成跟进漏斗和“二刷”名单，不错过任何一个潜在的合作机会。

商业级安全防护：集成密码鉴权和密钥隔离，保护核心业务资产。

✨ 核心功能模块
功能	描述	PM 业务视角
📥 智能提取与录入	允许用户粘贴 YouTube/IG 描述栏等乱序文本，利用正则表达式自动清洗并提纯有效邮箱地址，秒级存入云端数据库。	解决实习生录入慢、易出错的痛点。

🛠 技术架构 (Tech Stack)
项目采用了现代化的微型 SaaS (Micro-SaaS) 架构，实现了极速开发与部署：

前端交互 (Front-end): Streamlit — 极速打造数据应用，纯 Python 实现响应式 UI。

后端服务 (Back-end): Supabase — 商业级 Firebase 替代方案，提供可靠的云端 PostgreSQL 数据库与 API 服务。

数据处理 (Data Processing): Pandas & Re — Python 强大的正则表达式库用于数据清洗，Pandas 用于看板数据分析。

托管与部署 (Hosting & CI/CD): GitHub & Streamlit Cloud — 代码托管与云端持续集成/持续部署流水线。

🛡️ 安全规范 (Security)
作为商业工具，项目遵循严谨的安全红线标准：

密钥隔离 (Secrets Management): 数据库 URL、Key 以及应用访问密码严禁硬编码在代码中。全部资产均锁在 Streamlit 云端后台的 Secrets 变量中。即使 GitHub 仓库设为公开，核心钥匙依然绝对安全。

密码保护 (Access Control): 任何用户（包括同事和朋友）打开网址后，必须通过团队内部设置的 APP_PASSWORD 鉴权后，方可查看到具体的业务功能和数据库数据。
